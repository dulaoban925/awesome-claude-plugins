临时切换 Claude Code 的回复语言（仅在当前会话中生效）。

**用法**：
```
/lang <语言代码>
```

**支持的语言代码**：
- `zh-CN`: 简体中文
- `en`: 英语
- `ja`: 日语
- `zh-TW`: 繁体中文
- `ko`: 韩语
- `fr`: 法语
- `de`: 德语
- `es`: 西班牙语
- `ru`: 俄语

**示例**：
```bash
/lang zh-CN    # 切换到中文
/lang en       # 切换到英语
/lang ja       # 切换到日语
```

**注意**：
- 此命令仅在当前对话会话中生效
- 要永久更改默认语言，请使用 `/lang-config set <语言代码>`
- 新会话将使用配置文件中的默认语言

**查看更多选项**：
```bash
/lang-config list    # 查看所有支持的语言
/lang-config show    # 查看当前配置
```
